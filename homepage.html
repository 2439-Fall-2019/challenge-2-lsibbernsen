Code Breaking and Ciphers

In cryptography, a cipher (or cypher) is a method for protecting data through encryption and decryption. Most ciphers require a specific key for encryption and decryption, but some ciphers like the ROT13 or Atbash ciphers have fixed key.

Purpose

The purpose of this site is to educate about ciphers for both practical and extracurricular use. By the end of this unofficial course, you will have knowledge of several different ciphers and code breaking techniques, with optional pages for further learning. While this site is structured in a linear format, you may skip to any page you want using the master page.

Note: All codes are lowercase only
